## 2. ✅ What was fixed today (code, committed)

### 🔴 Critical (was blocking real developer onboarding)
- **Compliance documents now actually upload.** `ProjectForm` uploads every attached
  file to the new `project-documents` bucket and stores the real public URL, so
  verifiers/buyers can open them. ([ProjectForm.vue](../src/components/ProjectForm.vue),
  [storageService.js](../src/services/storageService.js) `uploadProjectDocument`)
- **Required documents are enforced.** A new project can no longer be submitted with
  missing required docs; a live "N required documents still needed" hint shows which.
  Edits preserve previously-uploaded docs.

### 🟠 Visible bugs
- **Verifier status badges** now render for the real workflow statuses
  (`submitted / in_review / needs_revision / validated`) — they were unstyled/blank before.
- **Verifier price-save no longer fails silently** — if the credit price can't be saved,
  validation aborts and the verifier is told (previously showed a false success).
- **Developer "Contact Support"** button now works (mailto support@carbonify.com).
- **Seller Earnings** now shows an error + retry state instead of silently showing zeros.
- **LGU data integrity** — diverted tonnage is clamped to ≤ generated on save and the
  form rejects diverted > generated, so diversion rate can never exceed 100%.

### 🟢 Polish + enhancements
- Brand green unified (removed off-brand `#10b981` from developer/verifier/LGU surfaces).
- **Role-aware landing** — developers land on `/developer/projects`, verifiers on
  `/verifier`, LGUs on `/lgu`, admins on `/admin` after login (was always `/home`).
- **Developer first-run empty state** — "Submit your first project" hero + `draft` badge styled.
- **Verifier**: "Validate" now warns if the rubric is incomplete; MRV **reject** requires a
  reason + confirm; MRV vintage defaults to the report period.
- **LGU**: records table scrolls on mobile; silent load failures now show an error+retry;
  endorsements gained a "Review project details" link + an optional decision note.
- Deleted dead `SimpleProjectApproval.vue`; removed noisy debug logging in `ProjectForm`.

## 3. 🧪 Test today (after applying migration 19)

**Project Developer**
1. Log in as a developer → you should land on **My Project Submissions**.
2. With no projects, you see the **"Submit your first project"** hero.
3. Submit a project. Try submitting with missing docs → blocked with a clear list.
4. Attach all required PDFs + submit → succeeds.
5. Open the project detail (or have a verifier open it) → **each document link opens the PDF**.

**Verifier**
1. Log in as a verifier → lands on the Verifier panel; queue rows show **coloured status badges**.
2. Open a submitted project → fill the rubric; try **Validate** with the rubric incomplete → you get a warning.
3. Set a price + Validate → project validates, price persists (no false success).
4. MRV review: try **Reject** with no note → blocked; with a note → confirms then rejects.

**LGU**
1. Log in as an LGU user → lands on **LGU Tools**.
2. MSW calculator: enter diverted > generated → rejected with a message.
3. Save a valid record → appears in Records; diversion rate ≤ 100%.
4. Endorsements: open "Review project details", add a note, Endorse/Decline.
